Task 3 - Firebase Integration and Core E-Commerce Features

As part of the Apex Planet Internship, Task 3 focused on integrating Firebase services into the ShopEase E-Commerce application. This task involved implementing Firebase Authentication for user login and registration, configuring Cloud Firestore for storing product and user data, and developing core e-commerce functionalities such as product display, cart management, wishlist handling, and order tracking. The project structure was further enhanced to support scalable and maintainable application development.

📌 Project Overview

ShopEase is a Flutter-based E-Commerce application developed as part of the Apex Planet Internship program. Task 3 focuses on implementing Firebase services and core e-commerce functionalities to create an interactive and scalable shopping application.
